# GigPay Tracker

GigPay Tracker is a full-stack application designed for gig workers, cab drivers, and delivery riders to track rides, calculate earnings, manage taxes, and generate monthly income reports.

## Features

* Ride Management (Create, Read, Update, Delete)
* Monthly Earnings Tracking
* Tax Calculation
* PDF Report Generation
* User Authentication
* Dashboard Analytics

## Tech Stack

### Backend

* Node.js
* Express.js
* Prisma
* PostgreSQL
* JWT Authentication

### Frontend

* React.js
* Tailwind CSS
* Axios
